## Story

**As a** administrador,
**I want** cadastrar, editar, curar e suspender parceiros (hotéis, pousadas, guias, restaurantes, etc.),
**so that** o catálogo de parceiros esteja completo para montar pacotes de viagem.

---

## Acceptance Criteria

1. [x] Listagem de parceiros com busca por nome, filtro por tipo e destino
2. [x] Formulário completo: destino, tipo, nome, descrição, contato, localização, preço, amenidades, imagens
3. [x] Criar novo parceiro (status pending por padrão)
4. [x] Editar parceiro existente
5. [x] Ação "Curar" — marca is_curated=true e contract_status=active
6. [x] Ação "Suspender/Reativar" — alterna contract_status
7. [x] Badges visuais para status (Pendente, Ativo, Suspenso) e curadoria
8. [x] Testes unitários para formulário e ações (9 testes)
9. [x] TypeScript sem erros
