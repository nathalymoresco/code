# Story 4.4 — Concierge IA Chat

**Epic:** 4 — Experiência & Acompanhamento
**Status:** Done
**Branch:** master

## Objetivo

Chat de concierge com IA para suporte 24/7 durante a viagem, com detecção de palavras de escalação e redirecionamento para atendimento humano via WhatsApp.

## Critérios de Aceite

- [x] FAB (Floating Action Button) para abrir chat
- [x] Painel de chat com histórico de mensagens
- [x] Mensagem de boas-vindas personalizada com destino
- [x] Input de texto e botão de envio
- [x] Detecção de keywords de escalação (emergência, cancelar, etc.)
- [x] Mensagem de escalação automática para suporte urgente
- [x] Botão de escalação para WhatsApp com mensagem pré-preenchida
- [x] Indicação de horários: IA 24/7 · Humano 8h-22h
- [x] Botão de fechar chat
- [x] API de chat com respostas rule-based (FAQ) para MVP
