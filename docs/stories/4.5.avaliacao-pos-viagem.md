# Story 4.5 — Avaliação Pós-Viagem

**Epic:** 4 — Experiência & Acompanhamento
**Status:** Done
**Branch:** master

## Objetivo

Formulário de avaliação pós-viagem com rating geral, avaliação por item/parceiro, NPS e comentários. Ratings de parceiros são atualizados automaticamente (média ponderada).

## Critérios de Aceite

- [x] Rating geral de 1 a 5 estrelas (obrigatório)
- [x] Avaliação individual por item (exclui tipo "seguro")
- [x] Nome do parceiro exibido em cada item avaliável
- [x] Escala NPS de 0 a 10
- [x] Campo de comentário geral
- [x] Submit desabilitado sem rating geral
- [x] Submit habilitado após selecionar rating
- [x] API salva review no pacote e atualiza rating dos parceiros
- [x] Tela de sucesso após envio
