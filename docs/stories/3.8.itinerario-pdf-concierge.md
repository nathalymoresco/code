## Acceptance Criteria

1. [x] API route gera HTML do itinerário: capa, dia a dia, contatos parceiros, Maps links, seguro, concierge
2. [x] Download disponível na página do pacote (botão "Baixar itinerário")
3. [x] WhatsApp Business do concierge com mensagem pré-preenchida (pacote + destino)
4. [x] Concierge humano no MVP (card com botão "Falar com concierge")
5. [x] Countdown na página do pacote (dias, horas, minutos até a viagem)
6. [x] Countdown mostra "Boa viagem!" quando data passou
7. [x] Testes: 12 cenários (4 countdown + 6 concierge + 2 download)
