## Story

**As a** sistema,
**I want** montar pacote automaticamente baseado em destino, datas e DNA,
**so that** o usuário receba proposta completa sem montar manualmente.

---

## Acceptance Criteria

1. [x] API Route `POST /api/packages/generate`: destination_id, dates, num_travelers, comfort_level
2. [x] Seleciona hospedagem por DNA (sociabilidade → hotel vs pousada)
3. [x] Distribui passeios por ritmo (alto → 2/dia, normal → 1/dia)
4. [x] Adiciona transfers ida/volta, alimentação, seguro
5. [x] Gera package_items com horários realistas
6. [x] Calcula preço total com markup 15%
7. [x] Status `draft` — ajustável pelo usuário
8. [x] Comfort level filtra price_range dos parceiros
9. [x] Testes: 20 cenários (dates, pricing, DNA personalization, validation)

## Technical Notes

- Insurance: R$12/pessoa/dia (básico, não removível)
- Transfers: não removíveis, dia 1 e último dia
- Atividades: distribuídas nos dias intermediários, removíveis
- Compatibility score obtido via RPC match_destinations
- Fallback: funciona mesmo sem parceiros (pacote mínimo)
